# 🤖 R.A.T - ROBOTIC REMOTE ADMIN TOOL

**Cyberpunk-themed RAT (Remote Administration Tool) with real hardware control via Web APIs.**

> "I just give the tools, whether they're used right or not is your business, boss."

---

## 🧠 Overview

This project allows you to **control another phone** (target) using a web-based controller panel.  
The target only needs to open a link — **no app installation required**.

### 🔥 Real Hardware Effects

| Command | Effect on Target's Device |
|---------|---------------------------|
| **Flashlight** | Turns on camera flash LED |
| **Vibrate** | Triggers vibration motor |
| **Battery** | Reads actual battery % |
| **Wake Lock** | Prevents screen from sleeping |
| **Orientation** | Locks screen to landscape |
| **Location** | Gets real GPS coordinates |
| **Notification** | Sends system notification |
